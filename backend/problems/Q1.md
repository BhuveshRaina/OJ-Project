# Problem Statement

**Given an array of integers, find the maximum subarray sum.**  
The subarray must be contiguous, and you need to return the largest possible sum from any subarray.

---

## Example Test Cases

### Example Test 1  
**Input:**  
[1, 2, 3, 4, 5]  
**Output:**  
15

---

### Example Test 2  
**Input:**  
[-2, 1, -3, 4, -1, 2, 1, -5, 4]  
**Output:**  
6
